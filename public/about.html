About & Contact

MAKE PIXEL DOT is a lightweight browser editor for pixel art, sprites, icons, tiles, and small frame animations.

업데이트: 2026년 6월 30일

소개 / 문의

MAKE PIXEL DOT은 픽셀 아트, 스프라이트, 아이콘, 타일, 짧은 프레임 애니메이션을 만들 수 있는 브라우저 기반 에디터입니다.

What the Site Provides

The editor includes drawing tools, palette management, layers, frame animation, timeline matrices, preview tools, and export options for PNG, GIF, spritesheet, and .dotproj project files.

Who It Is For

MAKE PIXEL DOT is designed for game creators, students, icon designers, and hobby artists who want a focused place to sketch small pixel assets without installing desktop software.

Local-First Workflow

Drawing work happens in the browser. Users can save .dotproj files locally and export final artwork as image files for games, websites, social posts, or personal projects.

제공하는 기능

에디터는 펜슬, 지우개, 채우기, 스포이드, 도형 도구, 팔레트 관리, 레이어, 프레임 애니메이션, 타임라인 매트릭스, 미리보기, PNG/GIF/스프라이트시트/.dotproj 내보내기 기능을 제공합니다.

대상 사용자

MAKE PIXEL DOT은 데스크톱 프로그램을 설치하지 않고 작은 픽셀 자산을 빠르게 만들고 싶은 게임 제작자, 학생, 아이콘 디자이너, 취미 창작자를 위해 만들어졌습니다.

로컬 우선 작업 방식

그림 작업은 브라우저 안에서 진행됩니다. 사용자는 .dotproj 파일을 로컬에 저장하고, 완성된 작업물은 게임, 웹사이트, 소셜 이미지, 개인 프로젝트에 사용할 이미지 파일로 내보낼 수 있습니다.
