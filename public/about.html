소개 / 문의

MAKE PIXEL DOT은 아티스트, 게임 개발자, 학생, 취미 창작자를 위한 브라우저 기반 픽셀 아트 에디터입니다.

What MAKE PIXEL DOT Does

MAKE PIXEL DOT helps users create pixel art, icons, game sprites, tiles, and small frame animations directly in a web browser. The editor includes drawing tools, a palette manager, layers, animation frames, timeline matrices, live previews, and export tools for PNG, GIF, spritesheet, and .dotproj project files.

Why This Site Exists

The goal of MAKE PIXEL DOT is to provide a focused, lightweight drawing space for quick sprite work. It is designed for people who want to sketch pixel ideas without installing desktop software, create small assets for games or websites, or learn the basics of pixel art through hands-on editing.

Local-First Editing

The editor is built around local browser workflows. Artwork is processed in the browser while you draw. Users can save project files locally and reopen them later. This keeps the drawing experience simple and gives users direct control over their own exported files.

MAKE PIXEL DOT이 제공하는 기능

MAKE PIXEL DOT은 브라우저에서 픽셀 아트, 아이콘, 게임 스프라이트, 타일, 작은 프레임 애니메이션을 만들 수 있도록 돕는 도구입니다. 연필, 지우개, 채우기, 스포이드, 도형, 명암 도구와 팔레트, 레이어, 프레임, 타임라인 매트릭스, 미리보기, PNG/GIF/스프라이트시트/.dotproj 내보내기를 제공합니다.

이 사이트의 목적

MAKE PIXEL DOT의 목표는 빠른 스프라이트 작업에 집중할 수 있는 가볍고 명확한 드로잉 공간을 제공하는 것입니다. 데스크톱 프로그램을 설치하지 않고 픽셀 아이디어를 스케치하거나, 게임과 웹사이트용 작은 에셋을 만들거나, 직접 편집하면서 픽셀 아트 기초를 익히고 싶은 사용자를 위해 설계되었습니다.

로컬 우선 편집

에디터는 브라우저 안에서 작업하는 흐름을 중심으로 만들어졌습니다. 그림을 그리는 동안 작업물은 브라우저에서 처리되며, 사용자는 프로젝트 파일을 로컬에 저장하고 나중에 다시 열 수 있습니다. 이 방식은 작업 경험을 단순하게 유지하고, 내보낸 파일을 사용자가 직접 관리할 수 있게 합니다.
