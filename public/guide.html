Pixel Art Editor Guide & FAQ

MAKE PIXEL DOT is a browser-based editor for drawing pixel art, game sprites, icons, tiles, and small frame animations. This guide explains the main workflow from a blank canvas to exported artwork.

Getting Started

The default canvas is designed for quick sprite work. A 32 x 32 canvas is small enough for icons and characters, but large enough to practice outlines, color blocking, highlights, and shadows.

Start by selecting the Pencil tool, choosing a foreground color from the palette, and drawing the main silhouette. Add a new layer for details or effects when you want to keep edits separate. If you are making an animation, add frames gradually and preview movement before polishing every pixel.

The editor runs in your browser. Project data is not intentionally uploaded while you draw, so save important work as a .dotproj project file and export final images as PNG, GIF, or spritesheet files.

Recommended first workflow

  1. Create or keep the default 32 x 32 canvas.
  2. Block out the large silhouette with the Pencil tool.
  3. Use Lighten and Darken to create simple shading.
  4. Add a second frame if you want idle or movement animation.
  5. Export PNG for still art or GIF for animation previews.

Tool Guide

Drawing Tools

  • Pencil: Paint individual pixels or drag to draw continuous strokes.
  • Eraser: Clear pixels back to transparency without changing other pixels.
  • Bucket: Fill a connected region of the same color or transparent area.
  • Eyedropper: Pick a color from the canvas into the active color slot.

Shape and Shading Tools

  • Line: Drag from a start point to an end point to draw a crisp pixel line.
  • Rectangle: Draw a rectangular outline for panels, tiles, UI, or props.
  • Ellipse: Draw circles and oval shapes for effects, coins, heads, and icons.
  • Lighten / Darken: Adjust painted pixels one step brighter or darker while keeping a pixel-art look.

Palette and Color Workflow

Pixel art often works best with a limited palette. Instead of using many similar colors, pick a small set of readable colors for outlines, midtones, highlights, and shadows. This makes sprites easier to recognize at small sizes.

Foreground

The main color used by drawing and shape tools.

Background

A secondary color slot for switching or sampling quickly.

Recent Colors

Recently used colors stay available for quick reuse while you work.

Selection Workflow

Use the Selection tool to drag out a rectangular selection.

Drag inside the selection to move only the selection box. Hold Shift while dragging to move the selected pixels with the box.

On touch devices, use the Move button in the selection toolbar instead of holding Shift.

Ctrl + C copies the selected pixels into the stamp buffer. A translucent preview shows what will be pasted at the current selection position.

Ctrl + V stamps the copied pixels, Ctrl + X cuts them, and Esc clears the selection.

Frames, Layers, and Animation

Timeline Matrix

Frames create motion. Add, duplicate, delete, and reorder frames to build simple loops. Matrices help organize separate animation sets such as Idle, Walk, Attack, Effects, or UI states.

Layers

Layers let you separate background, character, outline, effects, and details. This makes it easier to edit one part of a sprite without damaging the rest of the artwork.

FAQ

Q. Are my drawings uploaded to a server?

No. The editor is designed around local browser work. Save important art as .dotproj files or exported images.

Q. Can I use exported artwork commercially?

Artwork you independently create with the editor belongs to you. See the Terms of Service for more details.

Q. What are Lighten and Darken for?

They are shading tools for building highlights and shadows gradually without jumping straight to white or black.

Q. What should I export for a game engine?

Use PNG for individual sprites and spritesheet export when your engine expects multiple frames in one image.

Q. Does the editor work on mobile?

Yes. The mobile layout includes compact palette, tool, frame, and matrix controls for touch-based editing.

시작하기

기본 캔버스는 빠른 스프라이트 작업에 맞춰져 있습니다. 32 x 32 크기는 아이콘과 캐릭터를 만들기에 작고 명확하면서도, 외곽선, 색상 배치, 하이라이트, 그림자를 연습하기에 충분합니다.

Pencil 도구를 선택하고 팔레트에서 전경색을 고른 뒤, 먼저 큰 실루엣을 그려보세요. 디테일이나 이펙트를 분리하고 싶다면 새 레이어를 추가합니다. 애니메이션을 만들 때는 프레임을 조금씩 추가하고 움직임을 먼저 확인한 뒤 픽셀을 다듬는 것이 좋습니다.

에디터는 브라우저에서 실행됩니다. 그리는 동안 프로젝트 데이터는 의도적으로 서버에 업로드되지 않으므로, 중요한 작업은 .dotproj 파일로 저장하고 완성 이미지는 PNG, GIF, 스프라이트시트로 내보내세요.

처음 사용 추천 흐름

  1. 기본 32 x 32 캔버스를 그대로 사용하거나 새 캔버스를 만듭니다.
  2. Pencil 도구로 큰 실루엣을 먼저 잡습니다.
  3. Lighten과 Darken으로 간단한 명암을 만듭니다.
  4. 대기 동작이나 움직임이 필요하면 두 번째 프레임을 추가합니다.
  5. 정지 이미지는 PNG로, 애니메이션 미리보기는 GIF로 내보냅니다.

도구 가이드

그리기 도구

  • Pencil: 픽셀을 하나씩 칠하거나 드래그해서 연속 선을 그립니다.
  • Eraser: 픽셀을 투명 상태로 지웁니다.
  • Bucket: 같은 색이나 투명 영역으로 연결된 부분을 채웁니다.
  • Eyedropper: 캔버스에서 색을 찍어 현재 색상 슬롯에 가져옵니다.

도형과 명암 도구

  • Line: 시작점에서 끝점까지 픽셀 선을 그립니다.
  • Rectangle: 패널, 타일, UI, 소품에 쓸 수 있는 사각형 윤곽선을 그립니다.
  • Ellipse: 이펙트, 동전, 머리, 아이콘 등에 쓸 원과 타원을 그립니다.
  • Lighten / Darken: 픽셀 아트 느낌을 유지하면서 칠해진 픽셀을 한 단계씩 밝게 또는 어둡게 조절합니다.

팔레트와 색상 작업

픽셀 아트는 제한된 팔레트에서 더 잘 읽히는 경우가 많습니다. 비슷한 색을 많이 쓰기보다 외곽선, 중간색, 하이라이트, 그림자에 사용할 작은 색상 세트를 정하면 작은 크기에서도 스프라이트가 더 명확하게 보입니다.

전경색

그리기와 도형 도구가 주로 사용하는 색상입니다.

배경색

빠르게 전환하거나 보조 색상으로 사용할 수 있는 색상 슬롯입니다.

최근 색상

최근 사용한 색상을 다시 쉽게 선택할 수 있습니다.

선택 도구 흐름

Selection 도구로 사각형 선택 영역을 만들 수 있습니다.

선택 영역 안을 드래그하면 선택 박스만 이동합니다. Shift를 누른 상태로 드래그하면 선택한 픽셀이 박스와 함께 이동합니다.

터치 기기에서는 Shift 대신 선택 툴바의 Move 버튼을 사용합니다.

Ctrl + C는 선택 픽셀을 복사합니다. 복사된 픽셀은 붙여넣기 전에 반투명 미리보기로 표시됩니다.

Ctrl + V는 붙여넣기, Ctrl + X는 잘라내기, Esc는 선택 해제입니다.

프레임, 레이어, 애니메이션

타임라인 매트릭스

프레임은 움직임을 만듭니다. 프레임 추가, 복제, 삭제, 순서 변경으로 간단한 루프를 만들 수 있고, 매트릭스는 Idle, Walk, Attack, Effect 같은 별도 애니메이션 세트를 정리하는 데 유용합니다.

레이어

레이어는 배경, 캐릭터, 외곽선, 이펙트, 디테일을 분리합니다. 한 부분을 수정할 때 다른 픽셀을 망가뜨리지 않아 작업을 더 안정적으로 진행할 수 있습니다.

FAQ

Q. 그림이 서버에 업로드되나요?

아니요. 에디터는 로컬 브라우저 작업 흐름을 중심으로 설계되었습니다. 중요한 작업은 .dotproj 파일이나 이미지로 저장하세요.

Q. 내보낸 작업물을 상업적으로 사용할 수 있나요?

사용자가 독립적으로 만든 작업물은 사용자에게 속합니다. 자세한 내용은 서비스 약관을 확인하세요.

Q. Lighten과 Darken은 무엇에 쓰나요?

하이라이트와 그림자를 단계적으로 만들기 위한 명암 도구입니다.

Q. 게임 엔진에 쓸 때는 무엇을 내보내야 하나요?

개별 스프라이트는 PNG를, 여러 프레임이 한 이미지에 필요하면 스프라이트시트를 사용하세요.

Q. 모바일에서도 사용할 수 있나요?

네. 모바일 레이아웃은 터치 편집을 위한 팔레트, 도구, 프레임, 매트릭스 컨트롤을 제공합니다.
