Getting Started

The default 32 x 32 canvas is a practical starting point for icons, characters, props, and animation tests. Start with a small silhouette, then refine the outline, color blocks, highlights, and shadows.

The editor runs in your browser. Save important work as a .dotproj project file when you want to continue editing later, and export finished artwork as PNG, GIF, or spritesheet files.

  1. Create or keep the default canvas.
  2. Choose a limited color palette.
  3. Draw the main shape with the Pencil tool.
  4. Add layers for details or effects.
  5. Preview animation frames before exporting.

Tool Guide

Drawing Tools

  • Pencil paints individual pixels or continuous strokes.
  • Eraser clears pixels back to transparency.
  • Bucket fills connected areas of the same color.
  • Eyedropper samples a color from the canvas.

Shape and Edit Tools

  • Line, Rectangle, and Ellipse help create clean pixel shapes.
  • Lighten and Darken adjust shading one step at a time.
  • Selection can copy, cut, paste, flip, delete, and move pixels.
  • Mirror options help build symmetrical sprites and icons.

Palette Workflow

Pixel art is easier to read when the palette is controlled. Pick a few colors for outline, midtone, shadow, and highlight instead of many similar shades. This keeps small sprites clear at game and icon sizes.

Frames, Layers, and Animation

Frames create motion. Duplicate a frame, change a few pixels, then preview the loop before polishing every detail.

Layers separate outlines, colors, highlights, shadows, backgrounds, and effects so one part can be edited without damaging the rest.

FAQ

Are my drawings uploaded?

No. The editor is designed around local browser work. Save your project files and exports yourself.

Can I use exported artwork commercially?

Artwork you independently create belongs to you. See the Terms of Service for details.

Does it work on mobile?

Yes. The editor includes compact controls for touch-based editing.

시작하기

기본 32 x 32 캔버스는 아이콘, 캐릭터, 소품, 애니메이션 테스트를 만들기 좋은 시작점입니다. 먼저 작은 실루엣을 잡고, 윤곽선과 색 블록, 하이라이트, 그림자를 차례로 다듬어 보세요.

에디터는 브라우저에서 실행됩니다. 나중에 계속 편집하려면 .dotproj 프로젝트 파일로 저장하고, 완성된 작업물은 PNG, GIF, 스프라이트시트로 내보낼 수 있습니다.

  1. 기본 캔버스를 사용하거나 새 캔버스를 만듭니다.
  2. 제한된 색상 팔레트를 선택합니다.
  3. 펜슬 도구로 큰 형태를 그립니다.
  4. 세부 요소나 효과는 레이어로 분리합니다.
  5. 애니메이션 프레임을 미리 확인한 뒤 내보냅니다.

도구 가이드

그리기 도구

  • 펜슬은 개별 픽셀이나 연속된 선을 그립니다.
  • 지우개는 픽셀을 투명 상태로 지웁니다.
  • 채우기는 같은 색으로 연결된 영역을 채웁니다.
  • 스포이드는 캔버스에서 색을 가져옵니다.

도형과 편집 도구

  • 선, 사각형, 원 도구는 깔끔한 픽셀 도형을 만드는 데 좋습니다.
  • 밝게와 어둡게는 명암을 한 단계씩 조절합니다.
  • 선택 도구는 복사, 잘라내기, 붙여넣기, 뒤집기, 삭제, 이동을 지원합니다.
  • 대칭 옵션은 좌우가 비슷한 스프라이트와 아이콘 제작에 유용합니다.

팔레트 작업

픽셀 아트는 팔레트를 제한할수록 작은 크기에서 잘 읽힙니다. 비슷한 색을 많이 쓰기보다 윤곽선, 중간색, 그림자, 하이라이트에 사용할 색을 소수로 정하면 게임 스프라이트와 아이콘 크기에서도 형태가 선명하게 보입니다.

프레임, 레이어, 애니메이션

프레임은 움직임을 만듭니다. 프레임을 복제하고 몇 픽셀만 바꾼 뒤 반복 재생을 확인하면 빠르게 애니메이션을 만들 수 있습니다.

레이어는 윤곽선, 색, 하이라이트, 그림자, 배경, 효과를 분리해 한 부분을 수정해도 다른 부분이 망가지지 않도록 돕습니다.

FAQ

그림이 서버로 업로드되나요?

아니요. 에디터는 로컬 브라우저 작업을 중심으로 설계되었습니다. 프로젝트 파일과 내보낸 이미지는 사용자가 직접 저장합니다.

내보낸 작업물을 상업적으로 사용할 수 있나요?

사용자가 독립적으로 만든 작업물은 사용자에게 있습니다. 자세한 내용은 서비스 약관을 확인해 주세요.

모바일에서도 사용할 수 있나요?

네. 모바일 화면에서는 터치 편집에 맞춘 압축된 도구, 팔레트, 프레임 컨트롤을 제공합니다.
